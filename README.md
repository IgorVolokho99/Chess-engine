# Chess-Engine MariaNetwork

This is a pet project where I am building a chess engine.

The most interesting part of this project is that I am developing it without using any AI tools.

## Modules
- Chess engine
- Website

## Technologies
- Python
- Flask
- Docker
- PostgreSQL

## UV install
```angular2html
curl -LsSf https://astral.sh/uv/install.sh | sh
```

```angular2html

```

```angular2html

```

```angular2html

```

```angular2html

```

```angular2html

```

```angular2html

```
